Sistema de Abastecimento

O primeiro projeto realizado, teve como objetivo desenvolver uma aplicação web para gerenciar o abastecimento dos veículos da empresa, otimizando o controle de gastos com combustível e facilitando o acompanhamento da utilização dos veículos. O sistema foi feito no estágio curricular e foram utilizadasa as linguagens: HTML, CSS, Javascript, PHP e MySQL.


Biblioteca Clarice Lispector

O meu segundo projeto, também realizado em estágio, foi para busca de livros na biblioteca Clarice Lispector. Os alunos podem pesquisar qualquer livro e conferir se o mesmo está disponível para empréstimo e também informações adicionais como a descrição do livro. O sistema é linkado com uma planilha com todas as informações e foi utilizado apenas HTML, CSS e Javascript para o projeto.
